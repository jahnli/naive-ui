# 浮动按钮 Float Button

跟 `Back Top` 很像，多一份颜值，多一份互动。

## 演示

```demo
basic.vue
badge.vue
```

## API

### Float Button Props

| 名称 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| bottom | `number \| string` | `40` | `Float Button` 距离页面底部的高度 |
| right | `number \| string` | `40` | `Float Button` 距离页面右侧的宽度 |
| to | `string \| HTMLElement` | `'body'` | `Float Button` 渲染的容器节点 |
